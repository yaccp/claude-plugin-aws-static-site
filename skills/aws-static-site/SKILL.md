# Aws Static Site

Deploy static sites to AWS S3 + CloudFront with custom domains and SSL

## When to Use

This skill applies when the user mentions keywords related to: aws, static, site.

## Quick Start

Invoke the plugin:
```
/aws-static-site:action
```

This single command provides an interactive menu for all operations.

## Configuration

Configuration stored in `.claude/yaccp/aws-static-site/config.json`.
